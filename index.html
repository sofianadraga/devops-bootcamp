Cloud & DevOps
Bootcamp

Практичне освоєння DevOps-підходів у рамках навчальної програми EPAM. Фокус на роботі з інструментами, автоматизації процесів та взаємодії з інфраструктурою.

Про практику

Метою практики було формування базових професійних компетенцій у сфері DevOps, що включає розуміння роботи операційних систем, автоматизації процесів та використання сучасних інструментів розробки.

Значна частина навчання була побудована навколо практичних завдань, що виконувались у середовищі командного рядка. Це дозволило не лише ознайомитися з теорією, а й отримати практичний досвід.

Ключові акценти:

• робота в Linux-середовищі
• автоматизація через Bash
• написання Python-скриптів
• контейнеризація Docker

Теми навчання

Linux Practice

Робота з файловою системою, процесами та правами доступу. Використання командного рядка як основного інструменту.

Bash Practice

Написання скриптів для автоматизації задач, робота з аргументами та змінними.

Python Practice

Створення скриптів для обробки даних, автоматизації та роботи з файлами.

Docker Practice

Основи контейнеризації, створення образів та запуск контейнерів.

Практичні завдання

У процесі навчання виконувалися практичні завдання, спрямовані на закріплення знань та формування навичок роботи з інструментами DevOps.

Результат

У результаті проходження практики було сформовано базові компетенції у сфері DevOps, що включають роботу з операційними системами, написання скриптів та використання контейнерів.

Отриманий досвід є фундаментом для подальшого професійного розвитку та роботи з сучасними ІТ-системами.
