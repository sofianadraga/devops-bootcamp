Cloud & DevOps Bootcamp

Практичне освоєння сучасних DevOps-технологій у межах навчальної програми EPAM. Фокус на реальних інструментах, автоматизації та роботі з середовищами виконання.

Linux Bash Python Docker

Про практику

Метою практики було формування базових професійних компетенцій у сфері DevOps, а також набуття актуальних знань і навичок роботи з сучасними технологіями.

Основний акцент був зроблений на практичному виконанні завдань, що дозволило не лише ознайомитися з теорією, а й отримати реальний досвід роботи з інструментами.

Ключові аспекти:
  • Робота в середовищі Linux
  • Автоматизація через Bash
  • Скрипти на Python
  • Контейнеризація Docker

Що відбувалося під час практики

Ознайомлення з Linux та роботою в командному рядку
Написання Bash-скриптів для автоматизації задач
Використання Python для обробки даних
Робота з Docker-контейнерами

Теми навчання

Linux Practice

Bash Practice
Написання скриптів та автоматизація процесів.
Python Practice

Docker Practice
Створення образів та запуск контейнерів.

Практичні завдання

У межах кожного модуля виконувалися практичні завдання, які дозволили закріпити отримані знання та сформувати навички роботи з інструментами DevOps.

Результат

У результаті проходження практики було сформовано базові навички роботи з Linux, Bash, Python та Docker.

Отриманий досвід є основою для подальшого розвитку в напрямку DevOps, а також дозволяє краще розуміти принципи автоматизації та роботи з сучасною ІТ-інфраструктурою.
