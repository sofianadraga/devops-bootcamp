Cloud & DevOps
Bootcamp

Практичне освоєння DevOps-технологій у межах програми EPAM. У процесі навчання основна увага приділялася роботі з Linux, автоматизації процесів за допомогою Bash та Python, а також використанню Docker для створення ізольованих середовищ виконання.

Про практику

Метою проходження практики було формування базових професійних компетенцій у сфері DevOps, що включає розуміння принципів роботи операційних систем, автоматизації процесів та взаємодії з інфраструктурою.

Практика проходила у форматі навчальних модулів, де теоретичні знання одразу закріплювалися виконанням практичних завдань у командному рядку. Такий підхід дозволив глибше зрозуміти логіку роботи інструментів.

⚡ Практичний підхід — більшість часу присвячена виконанню завдань
⚙️ Сучасні технології — Linux, Bash, Python, Docker
📈 Формування навичок — орієнтація на реальні сценарії використання

Теми навчання

Linux Practice

Робота з файловою системою, процесами, правами доступу та базовими командами операційної системи Linux.
Bash Practice

Написання скриптів для автоматизації процесів, використання змінних та аргументів.
Python Practice

Використання Python для створення утиліт, обробки даних і базової автоматизації.
Docker Practice

Основи контейнеризації, створення образів, запуск і управління контейнерами.

Практичні завдання

У межах кожного модуля виконувалися практичні завдання, що дозволили закріпити отримані знання та сформувати практичні навички роботи з інструментами DevOps.

Результат

У результаті проходження практики було сформовано базові навички роботи з Linux, Bash, Python та Docker.

Отриманий досвід дозволяє краще розуміти принципи автоматизації, роботи з інфраструктурою та підготовки середовищ виконання, що є ключовими аспектами сучасного DevOps-підходу.
