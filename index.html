Cloud & DevOps
Bootcamp

Практичне освоєння DevOps-технологій у межах програми EPAM. У процесі навчання основна увага приділялася роботі з Linux, автоматизації процесів за допомогою Bash та Python, а також використанню Docker для створення ізольованих середовищ виконання.

Про практику

Метою проходження практики було формування базових професійних компетенцій у сфері DevOps, що включає розуміння принципів роботи операційних систем, автоматизації процесів та взаємодії з інфраструктурою.

Практика проходила у форматі навчальних модулів, де теоретичні знання одразу закріплювалися виконанням практичних завдань у командному рядку. Такий підхід дозволив сформувати системне розуміння інструментів.

⚡ Практичний підхід — більшість часу практика
⚙️ Сучасні технології — Linux, Bash, Python, Docker
📈 Формування навичок — орієнтація на реальні задачі

Теми навчання

Linux Practice

Робота з файловою системою, процесами та командами.
Bash Practice

Написання скриптів та автоматизація процесів.
Python Practice

Створення скриптів і робота з даними.
Docker Practice

Контейнеризація, створення образів і запуск сервісів.

Практичні завдання

У межах кожного модуля виконувалися практичні завдання, що дозволило закріпити отримані знання та сформувати навички роботи з інструментами DevOps.

Результат

У результаті проходження практики було сформовано базові навички роботи з Linux, Bash, Python та Docker.

Отриманий досвід є основою для подальшого розвитку у сфері DevOps та роботи з сучасною ІТ-інфраструктурою.
