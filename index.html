Cloud & DevOps Bootcamp

Практичне занурення у світ DevOps: від командного рядка до контейнеризації

Про практику

Практика була спрямована на формування базових професійних компетенцій у сфері DevOps, а також набуття актуальних знань і навичок роботи з сучасними інструментами. Особлива увага приділялася практичному застосуванню технологій через виконання завдань у середовищі командного рядка.

Що відбувалося під час практики

У процесі навчання здійснювалося опрацювання теоретичних матеріалів і виконання практичних завдань. Основний акцент був зроблений на розумінні принципів роботи операційних систем, автоматизації процесів за допомогою скриптів та основ контейнеризації.

Теми навчання

Linux Practice
Робота з файловою системою, командами, процесами та середовищем Linux.
Bash Practice
Написання скриптів для автоматизації задач та обробки даних.
Python Practice
Використання Python для створення простих скриптів і роботи з даними.
Docker Practice
Створення образів, запуск контейнерів та ізоляція середовищ виконання.

Практичні завдання

У межах кожного модуля виконувалися практичні завдання, які включали роботу з командним рядком, написання скриптів та запуск контейнерів.

Результат

У результаті проходження практики було сформовано базові навички роботи з Linux, Bash, Python та Docker. Отримані знання створюють основу для подальшого розвитку в напрямку DevOps та роботи з сучасною ІТ-інфраструктурою.
