Cloud & DevOps Bootcamp

Практичне занурення у сферу DevOps у межах програми EPAM. Вивчення сучасних технологій, інструментів автоматизації та роботи з інфраструктурою.

Про практику

Метою проходження практики було формування базових професійних компетенцій у сфері DevOps, а також набуття актуальних знань і практичних навичок роботи з сучасними інструментами розробки та автоматизації.

Основний акцент був зроблений не лише на теоретичному матеріалі, а й на практичному виконанні завдань у середовищі командного рядка.

Що відбувалося під час практики

🔹 Ознайомлення з середовищем Linux та основами роботи в терміналі
🔹 Написання Bash-скриптів для автоматизації задач
🔹 Використання Python для обробки даних та створення скриптів
🔹 Робота з Docker: створення образів і запуск контейнерів

Теми навчання

Linux Practice

Робота з файловою системою, правами доступу, процесами та основними командами операційної системи Linux.

Bash Practice

Написання скриптів, автоматизація процесів, робота з змінними та аргументами командного рядка.

Python Practice

Використання Python для створення утиліт, обробки файлів і базових автоматизованих сценаріїв.

Docker Practice

Основи контейнеризації, створення образів, запуск та управління контейнерами.

Практичні завдання

У межах кожного модуля виконувалися практичні завдання, спрямовані на закріплення отриманих знань та формування практичних навичок роботи з інструментами DevOps.

Результати та висновки

У результаті проходження практики було сформовано базові навички роботи з Linux, Bash, Python та Docker.

Отриманий досвід дозволяє краще розуміти принципи автоматизації, роботи з інфраструктурою та підготовки середовищ виконання, що є ключовими елементами сучасного DevOps-підходу.
